# Day 7 - NumPy Mini Project

## Student Marks Analyzer

This project analyzes student marks using NumPy.

## Features:
- Calculates average marks
- Finds highest and lowest scores
- Counts passed and failed students
- Assigns grades (A, B, C, D, F)

## Concepts Used:
- NumPy arrays
- Mean, Max, Min
- Conditional logic

## Learning Outcome:
Applied NumPy in a real-world problem instead of just theory.
