# Day 4 - NumPy Marks Analyzer

## 📌 Project Overview
This project analyzes student marks using NumPy and generates a structured report.

## 🔄 Upgrade from Day 3
This is an improved version of the Day 3 NumPy Marks Analyzer.

### Improvements:
- Added percentage calculation
- Added pass/fail status
- Improved output formatting (table view)
- Enhanced overall logic and readability

## 🚀 Features
- Calculate average, highest, and lowest marks
- Assign grades (A, B, C, D)
- Calculate percentage
- Determine pass/fail status
- Identify topper
- Display formatted student report

## 🧠 Concepts Used
- NumPy arrays
- Mathematical operations (mean, max, min)
- Conditional logic
- Array processing
