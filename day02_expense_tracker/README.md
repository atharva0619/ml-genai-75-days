# Day 2 - Python Project

## Expense Tracker (CLI)

A simple command-line based expense tracker built using Python.

## Features
- Add expenses with amount and category
- View all recorded expenses
- Calculate total spending

## Concepts Used
- Functions
- Lists & Dictionaries
- Loops (while, for)
- Exception Handling (try-except)

## How to Run
1. Open terminal
2. Run the file:
   python expense_tracker.py
