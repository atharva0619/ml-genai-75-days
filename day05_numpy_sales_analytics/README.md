# Day 5 - NumPy Sales Analytics System

## 📌 Project Overview
This project is an advanced sales analytics system built using NumPy.  
It analyzes multi-dimensional sales data including days, products, and regions.

## 🚀 Features
- Takes user input for:
  - Number of days
  - Number of products
  - Number of regions
  - Product prices
  - Sales data
- Calculates:
  - Total units sold per product
  - Revenue per product
  - Total revenue
  - Daily revenue
  - Region-wise sales
- Identifies:
  - Best selling product
  - Best performing region
  - Best revenue day
  - Low performing products

## 🧠 Concepts Used
- NumPy arrays (1D, 3D)
- Axis-based operations
- Broadcasting
- Aggregation functions (sum, mean, argmax)
- User input handling
