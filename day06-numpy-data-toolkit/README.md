# Day 6 - NumPy Data Analysis Toolkit

A simple command-line tool built with NumPy to perform basic statistical analysis on numerical data.

## Features

* Manual input (space-separated values)
* CSV file input
* Mean, Median
* Standard Deviation, Variance
* Minimum and Maximum
* Correlation between two datasets

## Tech Stack

* Python
* NumPy

## Project Structure

```
day06-numpy-data-toolkit/
├── main.py
├── sample_data.csv
└── README.md
```

## How to Run

```
pip install numpy
python main.py
```

```
10 20 30 40 50
```
